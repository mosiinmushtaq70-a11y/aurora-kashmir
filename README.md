# Aurora Kashmir Dashboard 🌌
**Live Deployment Status: Ready for Streamlit Cloud**

This repository contains the full source code for the Aurora Kashmir dashboard, featuring a 3D interactive globe, real-time cosmic weather telemetry, and predictive visibility scoring.

**Repository:** [aurora-kashmir](https://github.com/mosiinmushtaq70-a11y/aurora-kashmir)

## Features
- **3D Interactive Globe**: Visualizes aurora hotspots worldwide and provides a high-fidelity focus on Kashmir.
- **Real-Time Telemetry**: Fetches solar wind and Kp index data directly from NOAA and NASA.
- **Predictive Engine**: Calculates aurora probability for Kashmir based on live geomagnetic conditions.
- **High-Fidelity UI**: Modern, dark-themed dashboard with "neon pulse" graphs and cinematic transitions.

## How to Run Locally
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app/main.py`
