# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-18

### Added
- Initial open-source release of AuroraLens.
- Dual-Stage XGBoost ML Engine for high-precision aurora forecasting.
- Real-time NOAA DSCOVR and NASA OMNI telemetry pipeline.
- High-fidelity glassmorphic landing page with 3D dial and global heatmap.
- Comprehensive documentation: README, ML Engine Spec, Contributing, and Security guidelines.
- GitHub Action CI workflow for automated linting and building.

### Fixed
- Fixed encoding issues in landing page headers.
- Hidden Next.js production announcer and portal badge for a cleaner UI.
- Corrected footer link wiring for GitHub and LinkedIn.
- Optimized stats bar (TrustBar) placement for better narrative flow.
